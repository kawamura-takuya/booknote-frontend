# Booknote - 読書ノートアプリ（Frontend）

Booknote は、読んだ本のメモや気づきを記録し、管理するための Web アプリケーションです。  
このリポジトリはフロントエンド部分（React + TypeScript + Vite）です。

---

## 技術スタック

- **React 19**
- **TypeScript**
- **Vite**
- **CSS（modern-css-reset 使用予定）**
- **デプロイ**：Vercel（予定）
- **外部API**：
  - Google Books API
  - Google OAuth2.0 認証
